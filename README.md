usage: to run on terminal :  go run . "text" shadow | cat -e  (the text is the text you will write covered with the double qoute and the shadow is just a bannerFIle type from the list of bannerFiles , so you can choose to write the name of any of the type of the bannerfile among the list there and the font will change according to it)
